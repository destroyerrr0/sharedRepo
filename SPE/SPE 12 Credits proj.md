## SAST — Static Application Security Testing

👉 Checks **source code** without running it

### Example:

- SQL injection risk
- Hardcoded secrets

### Tool:

- Semgrep

--------------------------------

## SCA — Software Composition Analysis

👉 Checks **libraries/dependencies**

### Example:

- Using vulnerable version of log4j

### Tool:

- OWASP Dependency Check
